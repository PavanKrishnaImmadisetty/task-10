# Handwritten Digit Classification using KNN

## Overview
This project uses the **K-Nearest Neighbors (KNN)** algorithm to classify handwritten digits using the **Sklearn Digits dataset**.  
The task focuses on understanding **distance-based learning** and the effect of different **K values**.

---

## Dataset
- Source: `sklearn.datasets.load_digits()`
- 1797 samples
- 8Ã—8 grayscale images
- Digits from 0 to 9

---

## Steps
- Loaded and explored the dataset
- Visualized digit images
- Applied train-test split
- Scaled features using StandardScaler
- Trained KNN with multiple K values
- Evaluated accuracy and confusion matrix
- Visualized predictions

- Python

- Matplotlib
